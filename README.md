## 🎮 2D Unity Game (2d game1)
A 2D game currently in development, built from the ground up using the Unity Engine. This project focuses on custom scripting, smooth animations, and interactive combat mechanics.

### Core Features
* *Player Mechanics:* Custom C# scripts handling responsive player movement, jumping, and physics.
* *Combat & Animations:* Dynamic combat actions driven by complex Unity Animator controllers to prevent animation lockups.
* *Health System:* Fully functioning HP and damage systems for the player entity.
* *Enemy Entities:* Scripted enemy behaviors and interactions.

### Technologies Used
* *Game Engine:* Unity
* *Programming Language:* C#
